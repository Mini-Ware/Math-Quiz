# Want to try?
![](https://github.com/Mini-Ware/Math-Quiz/blob/main/attempt.png)
1) Clone this repository
```shell
$ git clone https://github.com/Mini-Ware/Math-Quiz
```
2) Compile the file (Using any C compiler)
```shell
$ cd Math-Quiz
$ gcc quiz.c -o quiz
```
3) Run the compiled file
```shell
$ ./quiz
```
